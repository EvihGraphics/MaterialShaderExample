# External UE Rendering Knowledge Base

This directory contains curated knowledge notes drawn from external community articles, tutorials, and historical implementations of Order-Independent Transparency (OIT) and custom rendering pipelines in Unreal Engine. 

These documents serve as references and mappings to the actual UE5.7 source code, helping to prevent duplicated effort and providing context for our AVBOIT P2.6T implementation.

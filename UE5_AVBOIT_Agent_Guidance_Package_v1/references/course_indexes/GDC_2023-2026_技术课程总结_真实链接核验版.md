# 2023

年度目录：[GDC Vault — GDC 2023](https://gdcvault.com/browse/gdc-23/)

2023 年的技术主线是：现代实时渲染方法继续向开放世界、移动端和跨世代项目扩展；天气、天空、水体、积雪和森林等环境系统开始形成可复用的工业级模块；机器学习与程序化工具逐步进入内容生产管线。

| 7 | [Grappling with Performance: Rendering Optimization Strategies in ‘Rumbleverse’](https://gdcvault.com/play/1029151/Grappling-with-Performance-Rendering-Optimization) | 通过实际项目说明如何识别 GPU/CPU 瓶颈、削减过度绘制、材质和灯光成本，并在不破坏画面目标的前提下制定优化优先级。 | **中高** |
| 8 | [Modernizing Rendering at Supercell](https://www.gdcvault.com/play/1029242/Modernizing-Rendering-at) | 介绍移动游戏渲染架构现代化过程，关注兼容性、性能、工具链以及旧内容迁移。对长期维护自定义渲染功能和避免一次性原型很有帮助。 | **中高** |

| 15 | [Visual Effects Summit: Navigating ‘The Callisto Protocol’’s VFX](https://gdcvault.com/play/1028823/Visual-Effects-Summit-Navigating-The) | 复盘写实科幻环境和战斗 VFX 的制作方法、照明融合、材质及性能约束。可用于研究烟尘、蒸汽、碎屑与场景光照的统一。 | **中高** |

# 2024

年度目录：[GDC Vault — GDC 2024](https://gdcvault.com/browse/gdc-24/)

2024 年的明显变化是：光追、GPU 驱动、GPU Work Graphs、超大世界和程序化内容不再停留在功能展示，而开始围绕 shipped 项目的内存、资产、工具与跨平台约束展开。

# 建议学习路线

1. **天气与体积基底**：GT7 Sky → Hogwarts Legacy → Borderlands 4 Clouds / Weather。
2. **局部模拟与环境交互**：Open-World Water → Alan Wake 2 GPU Vegetation → CPU/GPU Particles → Constructive Destruction。
3. **地表反馈**：Snow Rendering → Diablo IV H2O → Biome Materials / Terrain Technology。
4. **UE 底层与 Compute**：Hi-Fi RUSH Custom Renderer → Third-Party UE5 Integration → Shaders 501 → GPU Work Graphs。
5. **性能落地**：Mobile Rendering → DX12 Memory → Automated Shader Tests → Continuous Performance Monitoring → Metaperf。

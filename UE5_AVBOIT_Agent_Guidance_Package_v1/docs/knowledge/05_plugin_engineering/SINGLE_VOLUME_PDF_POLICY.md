# Single Volume 资料使用政策

## 可使用

`sources/pdf/在UE5外部模块中扩展复杂渲染管线实践.pdf`

仅用于学习：

- 为什么把复杂渲染能力产品化为插件；
- 功能分层和数据边界；
- Game Thread / Render Thread 分离；
- 调试、统计、跨版本维护；
- 网格阶段与全屏合成阶段的宏观职责。

## 明确禁止

- `SingleVolume.zip`；
- 任何 SingleVolume C++、Shader 和资源源码；
- 类／函数调用链复刻；
- 把体积 Ray March 实现移植进 AVBOIT；
- 通过其他路径重新下载该源码补入指导包。

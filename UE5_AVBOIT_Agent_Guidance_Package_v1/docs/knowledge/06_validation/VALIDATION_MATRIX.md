# Validation Matrix

| 层级 | 场景 | 主要问题 | 参考标准 |
|---|---|---|---|
| V0 | 未修改 `15_Transparency` | 实验场是否可重复 | Release 1.58 基线 |
| V1 | 三张非交叉 RGB 面片 | Draw 顺序无关性 | Sorted Alpha / CPU Over |
| V2 | 多层非交叉卡片 | 深度复杂度与误差 | 高精度排序参考 |
| V3 | 交叉卡片 | 对象排序失效 | 逐像素或高精度参考；不得盲信对象级 Sorted Alpha |
| V4 | 动态沙暴 Flipbook | 时间稳定性 | 固定相机／运动相机序列比较 |
| V5 | 压力场景 | 性能和显存 | 同一内容、分辨率、API、采样条件 |

每轮必须记录：场景、相机、分辨率、API、GPU、提交顺序、算法参数、截图、GPU 时间和显存。

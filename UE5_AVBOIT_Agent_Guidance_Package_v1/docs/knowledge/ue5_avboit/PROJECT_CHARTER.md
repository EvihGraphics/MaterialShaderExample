# UE5 AVBOIT Project Charter

## 当前产品

**UE5.7 AVBOIT Unit Test 插件**。

## 当前目标

在 MaterialShaderExample 的真实插件工程上，建立一个可独立验证的 AVBOIT 后端，并通过固定 RGB 面片、六种提交顺序、SceneDepth 遮挡、线性颜色读回与自动误差指标证明其正确性。

## 后续产品

1. 插件私有 Card 输入；
2. Flipbook 沙暴片播放器；
3. Niagara Adapter；
4. 更深的 UE 半透明材质与 Mesh Pass 接入。

## 当前非目标

- 完整接管 UE 所有半透明材质；
- Niagara；
- Substrate 通用透明接入；
- 折射、DOF 前后半透明、多 View、VR 和移动端；
- 完整沙暴产品化；
- Engine Fork，除非有证据证明插件路径不可达。

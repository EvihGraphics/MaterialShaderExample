# Single Volume PDF Usage Guide

## 允许资料

`references/pdf_md/在UE5外部模块中扩展复杂渲染管线实践.md`

## 可吸收内容

- SceneViewExtension 回调；
- Renderer 委托；
- Component / SceneProxy / ViewExtension 分层；
- Game Thread 与 Render Thread 边界；
- MeshBatch、VertexFactory 和 Material Shader 思路；
- RDG / Compute / Full-screen Composite；
- 插件部署、调试和监控。

## 禁止

- 使用 `SingleVolume.zip`；
- 根据 ZIP 补全 PDF；
- 复制 Single Volume 具体类结构作为 AVBOIT 固定结构；
- 把体积 Ray March 算法当成 AVBOIT 算法。
